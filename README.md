## Description

Please find the interface inside src->types folder

This service is depending upon redis for data persistance upto 5 hours. Please install redis seperately.

Find the postman collection in this repo

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
